# Doujin Map Marker

## CLI Commands

*   `npm install`: Installs dependencies
*   `npm run generate-data`, `npm run gen-data`: Create group data in JSON format
*   `npm run sanitize-data`, `npm run san-data`: Sanitize group data and output in JSON format (should run after `npm run generate-data`)
*   `npm run gen-san-data`: Generate and sanitize group data
*   `npm run dev`: Run a development server at http://localhost:5173/
*   `npm run build`: Production-ready build
*   `npm run build:gh`: Production-ready build for GitHub Pages
*   `npm run serve`: Run a production-like server at http://localhost:8080/
*   `npm run format`: Use Prettier to format the code
*   `npm run test`: Run test
